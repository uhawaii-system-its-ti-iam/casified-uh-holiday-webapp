# What's in here?

In the 'styles' folder, we store custom css stylesheets, which help maintain a consistent look and feel across an entire website. By defining styles centrally in a stylesheet, you ensure that elements like headings, paragraphs, and buttons have a uniform appearance. This way, we can minimize manual styling.
