# What's in here?

The 'Components' folder contains the building blocks of our pages. Rather than writing all our visual elements into each page, 
we use components to create small pieces of pages, which in some cases, will have more than one application. These are then imported
to the pages, where they are stitched together. This saves us both development time, and space. A couple of examples of commonly 
used components are the 'navbar' and the 'footer.'
